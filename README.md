# 🎯 Exam Management System (C)

A console-based exam system built in C.

## 🚀 Features
- Student login/signup
- Admin panel
- Add/Edit/Delete questions
- MCQ exam system
- Timer with countdown
- Negative marking
- Result & grade system
- Leaderboard

## 🛠 Language
- C Programming
- File Handling

## ▶️ How to Run
- Open in CodeBlocks
- Build & Run
